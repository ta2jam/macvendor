# IANA protocol-registry rights review

Review date: 2026-07-12

Next mandatory review: 2027-07-12

## Evidence

IANA and the IETF state that protocol registry data may be used by any party
for any purpose and apply the CC0 1.0 public-domain dedication:
<https://www.iana.org/help/licensing-terms>.

## Decision 2026-07-12

The EUI-48 unicast and multicast portions of the IANA Ethernet Numbers registry
are approved as `public_domain_claim` for derived `usage_note` API output.
They must not be presented as device-vendor assignments. Raw redistribution is
not part of this approval.
